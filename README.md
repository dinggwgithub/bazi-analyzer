# 八字分析系统 (BaZi Analyzer)

基于Go语言开发的专业八字分析系统，严格遵循《渊海子平》《滴天髓》等传统命理学典籍原则。

## 项目简介

本系统通过输入八字四柱（如：壬戌 壬寅 庚午 丙戌），即可按照专业命理分析流程，输出完整的八字分析报告。系统核心算法基于文档中详细阐述的6步分析流程，确保分析结果的专业性和准确性。

## 核心功能

- ✅ 八字解析与命盘构建
- 🔄 全局特殊关系扫描（天干五合四冲、地支三会三合六合）
- ⚡ 格局重构与合化判断
- 📊 日主旺衰分析
- 🎯 喜忌用神确定
- 🔍 干支作用关系细析
- 📈 格局高低综合论断
- 🌐 REST API 接口服务
- 📝 Markdown 格式报告输出
- 🧾 Swagger UI 接口文档

## 分析流程

系统严格遵循以下专业分析流程：

```
一看合冲，抓核心 → 二判成败，定格局 → 三论旺衰，辨喜忌 → 四察细节，断玄机
```

1. **第一步**：扫描全局特殊关系（天干五合四冲、地支三会三合六合）
2. **第二步**：格局重构（合化成功判断、五行重定）
3. **第三步**：定日主旺衰（得令、得地、得势分析）
4. **第四步**：明喜忌用神（根据旺衰确定喜忌）
5. **第五步**：细析作用关系（盖头截脚、刑冲害破、天干路线）
6. **第六步**：综合论断与格局判断

## 核心规则

### 特殊关系优先级
```
三会局 > 三合局 > 天干五合 > 地支六合 > 刑冲害破 > 普通生克
```

### 合化成功条件
- **得令**：合化神的五行正是出生月份的五行
- **透干**：合化神的五行在天干上出现
- **无克**：无强力阻克
- 规则：满足"得令"或"透干"之一，且无强力阻克，即合化成功

### 旺衰判断依据
- **得令**：日主是否生于生扶它的月份
- **得地**：日主在地支中是否有强根
- **得势**：天干是否有比肩、劫财帮助
- **克泄耗**：官杀、食伤、财星的综合压力

## 参考案例

### 案例：壬戌 壬寅 庚午 丙戌
- **扫描结果**：天干丙壬冲，地支寅午戌三合火局
- **合化判断**：寅午戌合化成功（火得令+透干）
- **旺衰分析**：庚金极弱，**从弱格**
- **喜忌用神**：喜火、水、木，忌金、土
- **格局评估**：从火格，格局清纯，层次较高
