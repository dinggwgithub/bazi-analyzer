# 八字分析系统 (Bazi Analyzer)

## 项目简介

基于 Go 语言开发的八字分析系统，根据传统八字命理学理论，对输入的四柱八字进行深度分析，输出详细的命理分析报告。

## 核心功能

- **四柱输入**：支持输入年、月、日、时四柱的天干地支（如：壬戌 壬寅 庚午 丙戌）
- **全局关系扫描**：自动识别天干五合、四冲、地支三会、三合、六合等特殊组合
- **格局重构**：根据合化条件进行五行重定，重构命局格局
- **日主旺衰判断**：综合得令、得地、得势分析日主旺衰
- **喜忌用神推导**：基于旺衰结论确定喜神、用神、忌神
- **作用关系细析**：详细分析干支组合、刑冲害破等微观关系
- **分析报告生成**：输出结构化的分析报告

## 技术栈

- **语言**：Go 1.21+
- **架构**：模块化设计，核心引擎 + 规则库

## 快速开始

```bash
# 克隆项目
git clone https://github.com/your-repo/bazi-analyzer.git

# 进入项目目录
cd bazi-analyzer

# 运行
go run main.go
```

## 使用示例

输入：
```
壬戌 壬寅 庚午 丙戌
```

输出：完整的八字分析报告，包括：
- 命局基本信息
- 全局特殊关系
- 格局重构结果
- 日主旺衰判断
- 喜忌用神
- 详细作用关系分析
