# 八字分析系统 (Bazi Analyzer)

一个基于 Go 语言开发的专业八字命理分析系统，严格遵循传统命理学经典理论，实现完整的八字分析流程。

## 项目概述

本系统根据《渊海子平》《滴天髓》《三命通会》《子平真诠》等经典命理典籍的理论基础，实现了完整的八字分析流程。用户只需输入八字四柱（如：壬戌 壬寅 庚午 丙戌），系统即可进行全面的命理分析。

## 核心功能

### 1. 基础数据管理
- 天干地支数据定义与管理
- 五行属性映射
- 十神关系计算
- 地支藏干系统

### 2. 全局特殊关系扫描
- **天干五合**：甲己合土、乙庚合金、丙辛合水、丁壬合木、戊癸合火
- **天干四冲**：甲庚冲、乙辛冲、丙壬冲、丁癸冲
- **地支三会局**：寅卯辰会木、巳午未会火、申酉戌会金、亥子丑会水
- **地支三合局**：申子辰合水、亥卯未合木、寅午戌合火、巳酉丑合金
- **地支六合**：子丑、寅亥、卯戌、辰酉、巳申、午未

### 3. 格局重构与合化判断
- 合化成功条件判断（得令、透干、无克）
- 五行重定与干支重评
- 能量聚焦与优先级处理

### 4. 日主旺衰分析
- 得令分析（月令判断）
- 得地分析（地支根气）
- 得势分析（天干比劫）
- 克泄耗综合评估

### 5. 喜忌用神分析
- 根据旺衰确定喜忌神
- 用神选取与忌神识别
- 从格特殊处理

### 6. 详细关系分析
- 本柱干支关系（盖头、截脚、通根、泄气）
- 地支刑冲害破分析
- 天干作用路线分析
- 隔干通关与阻隔判断

### 7. 综合论断输出
- 格局高低判断
- 命局特点总结
- 结构化分析报告

## 分析流程

系统严格遵循以下分析流程：

```
第一步：扫描全局特殊关系（直抓核心）
    ↓
第二步：格局重构（能量重定）
    ↓
第三步：定日主旺衰（于重构之基）
    ↓
第四步：明喜忌用神（战略决策）
    ↓
第五步：细析作用关系（微观侦查）
    ↓
第六步：断格局高低与综合论断
```

## 使用方式

### 命令行使用

```bash
# 分析八字
bazi-analyzer analyze "壬戌 壬寅 庚午 丙戌"

# 输出JSON格式
bazi-analyzer analyze "壬戌 壬寅 庚午 丙戌" --format json

# 详细分析模式
bazi-analyzer analyze "壬戌 壬寅 庚午 丙戌" --verbose
```

### 作为库使用

```go
package main

import (
    "fmt"
    "github.com/yourusername/bazi-analyzer/pkg/analyzer"
)

func main() {
    // 创建分析器
    a := analyzer.New()
    
    // 解析八字
    bazi, err := a.Parse("壬戌 壬寅 庚午 丙戌")
    if err != nil {
        panic(err)
    }
    
    // 执行分析
    result := a.Analyze(bazi)
    
    // 输出结果
    fmt.Println(result.String())
}
```

## 核心原则

本系统严格遵循以下核心原则：

1. **动静原则**：天干主动，主事象发生；地支主静，主能量深浅
2. **关系优先原则**：特殊关系的力量与优先级远高于普通生克
3. **能量聚焦原则**：合会局成立后，内部干支的局部关系被整体吸收
4. **远近原则**：作用优先级为 本柱 > 相邻柱 > 隔柱 > 遥柱

## 优先级规则

```
三会局 > 三合局 > 天干五合 > 地支六合 > 刑冲害破 > 普通生克
```
