# 八字分析系统开发需求文档

## 1. 项目背景

### 1.1 项目目标
开发一个基于 Go 语言的专业八字命理分析系统，严格遵循传统命理学经典理论，实现完整的八字分析流程。

### 1.2 输入输出规范
- **输入**：用户的八字四柱，格式为 "壬戌 壬寅 庚午 丙戌"
- **输出**：完整的八字分析报告，包含格局分析、旺衰判断、喜忌用神、详细关系等
- **说明**：系统无需出生时间推算，直接基于给定的四柱进行分析

### 1.3 参考文档
- [bazi_analyzer.md](../bazi_analyzer.md) - 八字分析详细总结文档

## 2. 功能需求

### 2.1 基础数据管理

#### 2.1.1 天干数据
- 定义十天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
- 五行属性映射
- 阴阳属性映射
- 序号索引（用于计算）

#### 2.1.2 地支数据
- 定义十二地支：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥
- 五行属性映射
- 阴阳属性映射
- 序号索引（用于计算）
- 藏干定义（本气、中气、余气）

#### 2.1.3 五行数据
- 五行定义：木、火、土、金、水
- 生克关系定义
- 旺相休囚死状态定义

### 2.2 八字解析功能

#### 2.2.1 四柱解析
- 输入字符串解析（如 "壬戌 壬寅 庚午 丙戌"）
- 转换为结构化数据（年柱、月柱、日柱、时柱）
- 日主提取（日柱天干）
- 月令提取（月柱地支）

#### 2.2.2 数据验证
- 天干合法性验证
- 地支合法性验证
- 输入格式验证

### 2.3 十神计算功能

#### 2.3.1 十神定义
根据日主与其他天干的生克关系，计算十神：
- 比肩、劫财（同我）
- 食神、伤官（我生）
- 偏财、正财（我克）
- 偏官、正官（克我）
- 偏印、正印（生我）

#### 2.3.2 十神判定规则
```
同我（同阴阳为比肩，异阴阳为劫财）
我生（同阴阳为食神，异阴阳为伤官）
我克（同阴阳为偏财，异阴阳为正财）
克我（同阴阳为偏官/七杀，异阴阳为正官）
生我（同阴阳为偏印/枭神，异阴阳为正印）
```

### 2.4 全局特殊关系扫描

#### 2.4.1 天干关系扫描
- **天干五合检测**：甲己、乙庚、丙辛、丁壬、戊癸
- **天干四冲检测**：甲庚、乙辛、丙壬、丁癸
- 合化条件判断（得令、透干、无克）

#### 2.4.2 地支关系扫描
- **地支三会局检测**：
  - 寅卯辰会木（东方木局）
  - 巳午未会火（南方火局）
  - 申酉戌会金（西方金局）
  - 亥子丑会水（北方水局）
  
- **地支三合局检测**：
  - 申子辰合水
  - 亥卯未合木
  - 寅午戌合火
  - 巳酉丑合金
  
- **地支六合检测**：
  - 子丑合土
  - 寅亥合木
  - 卯戌合火
  - 辰酉合金
  - 巳申合水
  - 午未合土

- **地支六冲检测**：
  - 子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲

- **地支相刑检测**：
  - 无礼之刑：子卯
  - 恃势之刑：寅巳申
  - 无恩之刑：丑未戌
  - 自刑：辰辰、午午、酉酉、亥亥

- **地支六害检测**：
  - 子未、丑午、寅巳、卯辰、申亥、酉戌

- **地支六破检测**：
  - 子酉、寅亥、辰丑、午卯、申巳、戌未

### 2.5 格局重构功能

#### 2.5.1 合化成功判断
判断合化成功的关键条件：
- **得令**：合化神的五行正是出生月份的五行（当令之气）
- **透干**：合化神的五行在天干上出现（引导）
- **无克**：无强力阻克

满足"得令"或"透干"之一，且无强力阻克，即可断合化成功。

#### 2.5.2 五行重定
- 天干五合成功后，参与五合的两个天干五行属性重定为合化后的五行
- 地支三合/三会成功后，参与合会的所有地支五行属性重定为合化后的五行
- 地支六合成功后，参与六合的地支五行属性重定为合化后的五行

#### 2.5.3 干支重评
- 重新评估天干及其坐支的状态
- 重新评估坐在合会地支上的天干根基变化

#### 2.5.4 优先级处理
处理冲突与交集的优先级：
```
三会局 > 三合局 > 天干五合 > 地支六合 > 刑冲害破 > 普通生克
```

### 2.6 日主旺衰分析

#### 2.6.1 得令分析
- 判断日主是否生于生扶它的月份
- 注意月令可能因合化而五行重定

#### 2.6.2 得地分析
- 判断日主在地支中是否有强根（禄、旺）
- 判断是否有印星生扶
- 注意根气可能因合化而增强或丧失

#### 2.6.3 得势分析
- 判断天干是否有比肩、劫财来帮助
- 注意比劫可能因天干五合而被合走

#### 2.6.4 克泄耗分析
- 计算官杀对日主的压力
- 计算食伤对日主的泄耗
- 计算财星对日主的消耗

#### 2.6.5 旺衰结论
综合以上分析，给出最终旺衰结论：
- 身旺
- 身弱
- 从强
- 从弱

### 2.7 喜忌用神分析

#### 2.7.1 喜忌判定规则
基于旺衰结论确定喜忌神：

**身弱**：
- 喜：生扶（印、比劫）
- 忌：克泄耗（官杀、食伤、财星）

**身旺**：
- 喜：克泄耗（官杀、食伤、财星）
- 忌：生扶（印、比劫）

**从格**：
- 顺其气势，气势为何五行，便喜何五行

#### 2.7.2 用神选取
- 确定用神（对日主最有利的五行）
- 确定喜神（辅助用神的五行）
- 确定忌神（对日主不利的五行）
- 确定闲神（影响不大的五行）

### 2.8 详细关系分析

#### 2.8.1 本柱干支关系
分析每一柱的天干与地支关系：
- **泄气**：干生支（如甲午，木生火）
- **得益**：支生干（如癸酉，金生水）
- **盖头**：干克支（如庚寅，金克木）
- **截脚**：支克干（如壬戌，土克水）
- **通根**：干支同类（如甲寅，木+木）

#### 2.8.2 地支特殊关系
分析剩余的刑、冲、害、破关系，代表命局内部的矛盾和压力点。

#### 2.8.3 天干作用路线
- 相邻作用 > 隔干作用 > 遥干作用
- 隔干通关与阻隔分析
- 通关：中间干起到连续相生的作用
- 阻隔：中间干起到阻断或压制的作用

### 2.9 综合论断功能

#### 2.9.1 格局高低判断
结合所有信息，判断格局层次：
- 分析核心矛盾点
- 分析十神组合
- 判断格局清纯或混杂

#### 2.9.2 分析报告生成
生成结构化的分析报告，包含：
- 八字基本信息
- 全局特殊关系汇总
- 格局重构结果
- 日主旺衰分析
- 喜忌用神分析
- 详细关系分析
- 格局高低论断
- 命局特点总结

## 4. 接口需求

### 4.1 命令行接口

```bash
# 基本分析
bazi-analyzer analyze "壬戌 壬寅 庚午 丙戌"

# JSON输出
bazi-analyzer analyze "壬戌 壬寅 庚午 丙戌" --format json

# 详细模式
bazi-analyzer analyze "壬戌 壬寅 庚午 丙戌" --verbose

# 帮助信息
bazi-analyzer --help
```

### 4.2 程序接口

```go
// 创建分析器
analyzer := NewBaziAnalyzer()

// 解析八字
bazi, err := analyzer.Parse("壬戌 壬寅 庚午 丙戌")

// 执行完整分析
result := analyzer.Analyze(bazi)

// 分步分析
specialRelations := analyzer.ScanSpecialRelations(bazi)
reconstructed := analyzer.ReconstructPattern(bazi, specialRelations)
wangshuai := analyzer.CalculateWangshuai(reconstructed)
xiyong := analyzer.DetermineXiYong(wangshuai)
relationships := analyzer.AnalyzeRelationships(reconstructed, xiyong)
```
