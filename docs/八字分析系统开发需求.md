# 八字分析系统开发需求

## 一、项目背景

开发一个基于 Go 语言的八字分析系统，根据传统八字命理学理论，对用户输入的四柱八字进行深度分析，输出详细的分析报告。

## 二、输入输出定义

### 2.1 输入

用户输入格式：四柱八字（年柱、月柱、日柱、时柱）

示例输入：
```
壬戌 壬寅 庚午 丙戌
```

说明：
- 每柱由一个天干和一个地支组成
- 天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
- 地支：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

### 2.2 输出

系统输出完整的八字分析报告，包含以下内容：

1. **命局基本信息**
   - 四柱展示
   - 五行统计
   - 十神定位

2. **全局特殊关系扫描**
   - 天干五合
   - 天干四冲
   - 地支三会局
   - 地支三合局
   - 地支六合
   - 次级关系记录

3. **格局重构结果**
   - 合化判断
   - 五行重定
   - 重构后命盘

4. **日主旺衰判断**
   - 得令分析
   - 得地分析
   - 得势分析
   - 旺衰结论

5. **喜忌用神**
   - 喜神列表
   - 用神确定
   - 忌神列表

6. **细析作用关系**
   - 干支组合分析
   - 地支特殊关系
   - 天干作用路线

## 三、核心功能需求

### 3.1 天干作用分析

#### 3.1.1 天干五合
| 组合 | 合化结果 |
|------|----------|
| 甲己 | 土 |
| 乙庚 | 金 |
| 丙辛 | 水 |
| 丁壬 | 木 |
| 戊癸 | 火 |

**合化成功条件**：
1. 得令：合化神的五行是出生月份的五行
2. 透干：合化神的五行在天干上出现
3. 无克：无强力阻克
4. 满足"得令"或"透干"之一，且无强力阻克

#### 3.1.2 天干四冲
| 组合 | 性质 |
|------|------|
| 甲庚 | 阳冲阳 |
| 乙辛 | 阴冲阴 |
| 丙壬 | 阳冲阳 |
| 丁癸 | 阴冲阴 |

### 3.2 地支作用分析

#### 3.2.1 三会局（力量最强）
| 组合 | 会化结果 |
|------|----------|
| 寅卯辰 | 木 |
| 巳午未 | 火 |
| 申酉戌 | 金 |
| 亥子丑 | 水 |

#### 3.2.2 三合局
| 组合 | 合化结果 |
|------|----------|
| 申子辰 | 水 |
| 亥卯未 | 木 |
| 寅午戌 | 火 |
| 巳酉丑 | 金 |

#### 3.2.3 六合
| 组合 | 合化结果 |
|------|----------|
| 子丑 | 土 |
| 寅亥 | 木 |
| 卯戌 | 火 |
| 辰酉 | 金 |
| 巳申 | 水 |
| 午未 | 火/土 |

#### 3.2.4 其他关系
- 六冲：子午、丑未、寅申、卯酉、辰戌、巳亥
- 三刑：寅巳申、丑未戌、子卯、辰午酉亥（自刑）
- 六害：子未、丑午、寅巳、卯辰、申亥、酉戌
- 六破：子酉、寅亥、辰丑、午卯、申巳、戌未

### 3.3 优先级规则

**处理优先级**：
```
三会局 > 三合局 > 天干五合 > 地支六合 > 刑冲害破 > 普通生克
```

**作用路线**：
```
本柱 > 相邻柱 > 隔柱 > 遥柱
```

### 3.4 干支组合关系

| 关系类型 | 名称 | 说明 |
|----------|------|------|
| 干生支 | 泄气 | 天干力量泄于地支 |
| 支生干 | 得益 | 天干得地支生扶 |
| 干克支 | 盖头 | 天干压制地支 |
| 支克干 | 截脚 | 地支克制天干 |
| 干支同类 | 通根 | 天干力量增强 |

### 3.5 日主旺衰判断

#### 3.5.1 判断维度
1. **得令**：日主是否生于生扶它的月份
2. **得地**：日主在地支中是否有强根或印星生扶
3. **得势**：天干是否有比肩、劫财帮助
4. **克泄耗**：官杀、食伤、财星的综合压力

#### 3.5.2 旺衰类型
- 身旺
- 身弱
- 从强
- 从弱

### 3.6 喜忌用神

| 日主状态 | 喜 | 忌 |
|----------|-----|-----|
| 身弱 | 印、比劫 | 官杀、食伤、财星 |
| 身旺 | 官杀、食伤、财星 | 印、比劫 |
| 从格 | 顺其气势 | 逆其气势 |

## 四、分析流程

### 第一步：扫描全局特殊关系
- 扫描天干五合、四冲
- 扫描地支三会、三合、六合
- 记录次级关系

### 第二步：格局重构
- 判断合化是否成功
- 进行五行重定
- 重新评估干支状态

### 第三步：定日主旺衰
- 分析得令
- 分析得地
- 分析得势
- 综合判断旺衰

### 第四步：明喜忌用神
- 根据旺衰确定喜忌
- 确定用神

### 第五步：细析作用关系
- 分析干支组合
- 分析地支特殊关系
- 分析天干作用路线

### 第六步：生成分析报告

## 五、非功能需求

### 5.1 性能要求
- 单次分析响应时间 < 100ms
- 支持批量分析

### 5.2 可扩展性
- 规则可配置
- 支持新增分析维度

### 5.3 可维护性
- 代码结构清晰
- 完善的注释文档
- 单元测试覆盖

## 六、约束条件

1. 用户输入仅为四柱八字，无需出生时间推算
2. 分析基于传统命理学理论
3. 不涉及封建迷信内容，仅供学习研究
