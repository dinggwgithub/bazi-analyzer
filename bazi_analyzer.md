# 八字分析详细总结

## 总纲：核心原则

1.	动静原则：天干主动，主事象发生；地支主静，主能量深浅。分析时，干重生克合冲，支重刑冲合害。
￮	《渊海子平》云：“干动而有为，支静以待用。” 天干主外，显动态之事象；地支主内，藏静态之能量。
￮	比喻：天干如天象，流转不息，主事之发生；地支如地势，厚德载物，主气之深浅。

2.	关系优先原则：特殊关系（天干五合、四冲；地支三会、三合、六合、刑冲害破）的力量与优先级远高于普通生克。分析时，必先观其有无特殊组合。
￮	《滴天髓》所言：“阖关之情意，亦有向背。” 干支间的特殊关系，决定了其能量是相互开放流通（阖）还是彼此排斥封闭（关）。

3.	能量聚焦原则：当天干五合、地支三合、三会局成立时，应视其为一个整体能量单位，其内部干支的局部关系均被吸收，不再具备独立分析价值。
￮	比喻：当“寅午戌”三合火局成立，整个团队释放“燎原之火”，不再单独分析“寅午”半合等局部关系。同理，甲己合土成功，甲与己各自独立性消失，共化土性。

4.	远近原则（作用优先级）：由内而外，由近及远：本柱 > 相邻柱 > 隔柱 > 遥柱。
￮	此原则源于五行能量的辐射与衰减规律，邻近者感应最强。



## 第一章：分析流程总览

本流程是后续所有章节的纲领，请先深刻理解并记忆此框架。

### 第一步：扫描全局特殊关系（直抓核心）
•	操作：拿到八字，首先同步扫描天干与地支，寻找所有具备全局性影响力的特殊组合。
•	目的：第一时间识别出命局中能量最强大、最可能改变五行属性的“能量联盟”与“核心冲突点”。
•	详细步骤：
a.	扫描天干：
▪	五合：检查是否有甲己、乙庚、丙辛、丁壬、戊癸的组合。
▪	四冲：检查是否有甲庚、乙辛、丙壬、丁癸的组合。
b.	扫描地支：
▪	三会局：检查是否有寅卯辰、巳午未、申酉戌、亥子丑的组合。
▪	三合局：检查是否有申子辰、亥卯未、寅午戌、巳酉丑的组合。
▪	六合：检查是否有子丑、寅亥、卯戌、辰酉、巳申、午未的组合。
c.	记录次级关系：如半三合（寅午）、拱合（寅戌）等。但此时仅作记录，不急于下结论，因为它们的最终命运取决于能否构成更强的全局性组合。

### 第二步：格局重构（能量重定）
•	IF (存在天干五合且合化成功) {
￮	执行：进行“天干五行重定”。
i.	五行重定：将参与五合的两个天干，其五行属性重定为合化后的五行（如甲己合化土，则甲木、己土均视为土）。
ii.	干支重评：重新评估这两个天干及其坐支的状态。合化成功则天干得“化神”之根气，合而不化则双方相互羁绊（合绊），力量均受牵制。
iii.	力量聚焦：停止分析被合化的两个天干之间的生克，以及它们与其他干支的独立生克关系（除非合绊解除）。转而分析‘化神’在整个命局中的新作用。
•	}
•	IF (存在地支三合/三会局且合化成功) {
￮	执行：进行“地支五行重定”。
i.	五行重定：将参与合会的所有地支，其五行属性重定为合化后的五行。
ii.	干支重评：重新评估坐在这些地支上的天干，其根基与状态因五行重定而发生的变化。
iii.	力量聚焦：停止分析该合会局内部各地支之间的任何局部关系（如半合、拱合、六合等）。
•	}
•	IF (存在地支六合且合化成功) {
￮	执行：进行“地支六合五行重定”，逻辑同三合三会，但力量层级较低。需注意，若参与六合的地支已被更强的三合/三会局“征用”，则六合不成立或被吸收。
•	}
•	ELSE {
￮	执行：维持原盘不变，继续下一步。
•	}

合化成功的关键条件（统一原则）：
•	得令：合化神的五行正是出生月份的五行（当令之气）。此为最重要条件。
•	透干：合化神的五行在天干上出现（引导）。
•	无克：对于天干五合，若合化双方被其他紧贴天干强力克制（如甲己合土，见紧贴乙木克己土），则合化不易成功。
•	满足“得令”或“透干”之一，且无强力阻克，即可断合化成功，力量极强。

处理冲突与交集的优先级：
•	原则：三会局 > 三合局 > 天干五合 > 地支六合 > 刑冲害破 > 普通生克。
•	操作：优先服从力量最强的组合。被“征用”的干支，不再参与力量较弱的组合。例如，一个地支同时参与三合和六合，优先服从三合局。

### 第三步：定日主旺衰（于重构之基）
•	操作：在 【格局重构后】 的命盘基础上（若无合会，则为原盘），论日主得令、得地、得势，定其旺衰。
•	输出：最终的、可靠的旺衰结论（身旺、身弱、从强、从弱）。
•	详细方法：
￮	看得令：日主是否生于生扶它的月份（注意：月令可能因合化而五行重定）。
￮	看得地：日主在地支中是否有强根（禄、旺）或印星生扶（注意：根气可能因合化而增强或丧失）。
￮	看得势：天干是否有比肩、劫财来帮助（注意：比劫可能因天干五合而被合走）。
￮	看克泄耗：计算官杀、食伤、财星对日主的综合压力。

### 第四步：明喜忌用神（战略决策）
•	操作：基于且仅基于 【第三步】 得出的最终旺衰结论，来确立喜神、用神、忌神。
•	详细规则：
￮	身弱：喜生扶（印、比劫），忌克泄耗（官杀、食伤、财星）。
￮	身旺：喜克泄耗（官杀、食伤、财星），忌生扶（印、比劫）。
￮	从格：顺其气势，气势为何五行，便喜何五行。

### 第五步：细析作用关系（微观侦查）
•	在明确的喜忌指导下，逐一分析未被合化格局吸收的干支组合、地支特殊关系、天干作用路线等细节。
•	具体包括：
￮	查干支组合：逐柱分析盖头、截脚、通根、泄气等情况。
￮	观地支特殊关系：寻找剩余的刑、冲、害、破，代表命局内部的矛盾和压力点。
￮	析天干作用路线：遵循“相邻作用 > 隔干作用 > 遥干作用”的路线，重点分析五合与相邻的克，以及隔干的“通关”与“阻隔”效应。

### 第六步：断格局高低与推岁运应期（综合论断与动态预测）
•	断格局：结合以上所有信息，看核心矛盾点与十神组合，判断格局层次。
•	推岁运：大运/流年干支是启动命局能量的“钥匙”。分析岁运时，重复第一步到第五步，但以原局为体，岁运为用，看其如何引动、改变原局平衡（尤其注意岁运触发原局天干五合、地支六合等）。



## 第二章：天干作用规律（干与干）

天干主天元，象征显性的、流通的、外在的事件与人物。其作用核心在于“动”与“通”。分析时必须首先处理天干五合与四冲。

### 2.1 基本作用形式

•	生、克：依据五行生克常态。（注意：在完成2.2节的优先级判断后，再分析剩余的普通生克）
•	五合：甲己合土、乙庚合金、丙辛合水、丁壬合木、戊癸合火。源于《素问·五运行大论》及《渊海子平·论十干合》。此为天干最高优先级的作用关系。
•	四冲：甲庚冲、乙辛冲、丙壬冲、丁癸冲。源于《三命通会·论冲击》。仅此四组，其激烈程度远超于“克”。优先级仅次于五合。
类型	组别	记忆口诀
生	甲乙→丙丁→戊己→庚辛→壬癸→甲乙	木火土金水循环
克	甲乙→戊己→壬癸→丙丁→庚辛→甲乙	木土水火金循环
合	甲己土、乙庚金、丙辛水、丁壬木、戊癸火	土金水木火，顺序相生
冲	甲庚、乙辛、丙壬、丁癸	阳冲阳，阴冲阴，戊己不冲

### 2.2 作用路线与力量层级

作用关系	作用力	规则详解与典籍依据
天干五合/四冲	绝对强（优先）
	核心规则：必须先判断天干五合与四冲。 一旦合化成功，天干属性改变；若合而不化，则形成“合绊”，双方均受牵制，难以再发挥原本作用。四冲则直接引发激烈冲突。《滴天髓·恩怨》：“两意情通中有媒，虽然遥立意寻追…” 即论合化之情与冲克之敌。
本柱天干	极强	天干与坐支关系最密，必先同观。干与支同柱，如人之居所，关系最密。分析天干，必先观其坐支，此为“根气”之所系。（此步在判断完五合/四冲后进行）
相邻天干	直接强	紧贴之干，可直接发生生、克。（此处的生克，指未被五合/四冲吸收的普通生克关系） 《子平真诠》在论“十干配合性情”时，皆以月、日、时相邻之干为例。此为天干作用之主干。
隔干	间接弱	核心规则：隔干不直接生克，但中间干起“通关”或“阻隔”作用。
- 通关：如年甲克日戊，月丙居中，则甲生丙，丙生戊，贪生忘克，化敌为友。《滴天髓》云：“关内有织女，关外有牛郎，此关若通也，相邀入洞府。”即指通关之妙。
- 阻隔：如年甲欲生日丙，月庚居中克甲，则甲生丙之力大减，此为“隔干压制”。分析隔干关系，本质是分析一个连续的作用链条。（此分析需建立在无五合/四冲干扰的前提下）
遥干	极微弱	年干与时干，距离最远。《三命通会》在论“论年月日时”中，均以日干为核心，逐次与月、年、时论关系，遥干之力多忽略。除非有合、冲关系，否则仅作五行气势之参考，无实质生克。（注意：遥干若有五合或四冲，仍需论其合冲，如年甲与时己，虽遥仍有相合之意，待岁运引动）

### 2.3 关键细节辨析（初学者必看）

•	关于“天干五合的优先级”：
￮	天干五合是最高优先级的组合，必须首先判断。合化成功则五行重定；合而不化则相互羁绊（合绊），大大减弱其生克他干的能力。合绊的双方，犹如两人手拉手，无法再去拿别的东西或攻击他人。
￮	案例：八字：甲寅 己巳 庚午 丙戌。年干甲木与月干己土紧贴相合。第一步即需判断甲己合化土是否成功。若生于辰戌丑未月或巳午月，土旺透干，则合化成功，年甲月己均化为土，年柱甲寅变为“土坐木”，甲木克寅木的关系被土的“盖头”所替代。若合化不成功，则甲木羁绊己土，己土生庚金之力大减。

•	关于“天干相冲”：
￮	必须严格限定为甲庚、乙辛、丙壬、丁癸四组。戊己土居中央，不参与对冲。
￮	与“克”的区别：冲的激烈程度远超于克。冲主决裂、变动、突发；克主制约、管理、压力。不可将“壬克丙”等误称为冲。
￮	优先级：仅次于五合，高于普通生克。
￮	冲 vs 克 精准区分
项目	冲	克
组别	仅四对：甲庚、乙辛、丙壬、丁癸	其余五行制约
性质	同性相斥，突发剧烈	异性制约，持续压力
应事	决裂、变动、突发	管理、压力、长期
举例	甲庚冲：车祸、打架	甲克戊：上级管下级

•	关于“隔干能否通关”：
￮	能，且有条件。通关五行必须能连续相生，且最终有益于日主或平衡全局。若通关后反而助长忌神，则为“传凶”，非真通关。
￮	前提：此分析必须在确定隔干之间无五合、四冲关系后进行。
￮	案例：年干：甲木 -> 月干：丙火 -> 日干：庚金
▪	甲木生丙火，丙火克庚金。丙火能通关吗？不能！ 因为甲木的力量通过丙火，最终去克伤了日主庚金，这是“放大忌神”而非通关。
▪	真通关：年甲→月丙→日戊甲生丙，丙生戊，化克为生，日主得益。
▪	假通关：年甲→月丙→日庚甲生丙，丙克庚，忌神放大，日主更伤。
•	关于“遥干能否作用”：
￮	基本不能直接生克。但若遥干构成五合或四冲，则此组合作为一种潜在关系存在，待岁运引动（如岁运地支构成合会局拉近其气，或岁运天干透出化神）时，会突然显现其影响力。
▪	条件：必须合化神得令+透干+中间无阻隔。
▪	案例：年甲时己，生于辰戌丑未月，土透干，可化土；否则仅“意向”。


## 第三章：地支作用规律（支与支）

地支主地元，象征隐性的、承载的、内在的能量与根基。其作用核心在于“情”与“势”。分析时必须遵循“三会 > 三合 > 六合 > 刑冲害破 > 普通生克”的优先级顺序。

### 3.1 基本作用形式

•	三会：寅卯辰会木、巳午未会火、申酉戌会金、亥子丑会水。（力量最强，最高优先级）
•	三合：申子辰合水、亥卯未合木、寅午戌合火、巳酉丑合金。（优先级次之）
•	六合：子丑、寅亥、卯戌、辰酉、巳申、午未。（优先级再次之）
•	冲：子午、丑未、寅申、卯酉、辰戌、巳亥。
•	刑：无礼之刑（子卯）、恃势之刑（寅巳申）、无恩之刑（丑未戌）、自刑（辰、午、酉、亥）。
•	害：子未、丑午、寅巳、卯辰、申亥、酉戌。
•	破：子酉、寅亥、辰丑、午卯、申巳、戌未。
•	普通生克：无特殊关系下的五行气势影响。（最低优先级，仅在无上述关系时参考）

### 3.2 作用路线与力量层级

作用关系	作用力	规则详解与典籍依据
三会/三合局	绝对强（优先）	核心规则：必须先判断三会、三合局。 成立后，其内部各地支间的所有局部关系均失去独立讨论意义。《滴天髓·同心》：“一方之气，不可不顾也。” 三合、三会即是“一方之气”，其力最专。

- 紧贴有关系：力量最强。
- 隔支有关系：力量存在，但减弱。

遥支	极微弱	年支与时支，若无三合、三会等全局性组合联系，其相互作用力可忽略不计，仅作远期、隐性参考。

### 3.3 关键细节辨析（初学者必看）

•	关于“三合三会局的优先性”：
￮	当命局构成三合局（如寅午戌合火）或三会局（如寅卯辰会木）时，此合会局的力量具有全局性和压倒性。
￮	规则：合会局一旦成立，其内部各地支之间的所有局部关系（包括六合、半合、拱合，甚至相邻生克）都失去独立讨论的意义，因为它们的力量已完全融入并服务于这个更强大的整体。应优先以合会局为一个整体能量团进行分析。

￮	命局同时出现：亥卯未三合木 + 寅卯辰三会木  
￮	判决：卯被三会局征用，三合局自动解散，亥 未退回普通支。
￮	
•	关于“六合的从属性”：
￮	六合的力量和成立条件，常常受到三合、三会局的影响。
￮	规则：若一个地支同时参与三合/三会和六合，优先服从三合/三会局。该地支与六合对象的关系被合会局吸收或切断。
￮	案例：八字中有申子辰三合水局，同时年支辰与时支酉有辰酉六合。此时，辰土作为水局之库，其土性已被水局转化，与酉金的六合关系让位于三合局，辰酉合不再成立，或力量可忽略不计。

•	关于“地支主静，无引不动”：
￮	理解：地支的能量通常是潜藏的，需要“钥匙”来启动。这把“钥匙”通常就是流年、大运的天干或地支。
￮	案例：命局中有酉金和辰土，但它们没有特殊关系。当流年天干透出庚/辛金时，就等于唤醒了辰中藏的乙木与酉中藏的辛金，辰酉之间就可能发生“辰酉合”的效应。（注意：若原局有更强合会，此合仍可能被抑制）

•	关于“三合局不存在争合”：
￮	核心规则：三合局是“三方媾和”，是一个团体行为，不存在“争合”。其逻辑是“众人拾柴火焰高”，多个相同地支参与，是增强了合局的力量和稳定性，而不是相互竞争。
￮	案例：八字：壬戌 壬寅 庚午 丙戌。年支戌和时支戌，都是寅午戌三合火局的参与者。这不是“两个戌土争着去合午火”，而是“两个戌土仓库共同为火局提供燃料”，使火局更强大、更稳固。


## 第四章：干支组合与藏干（能量关联）
这是分析八字能量的核心环节，决定了每个字的“生存环境”。
### 4.1 本柱干支五大关系
本柱干支：最直接、最亲密的相互作用
关系类型	名称	规则与影响
干生支	泄气	天干生地支。如甲午（木生火）。天干力量泄于地支，天干减力，地支增力。
支生干	得益	地支生天干。如癸酉（金生水）。天干得地支生扶，犹如树木有根，天干增力。日主得此，为得地。
干克支	盖头	天干克地支。如庚寅（金克木）。天干像盖子一样压制地支，地支力量受损严重。如伐木之斧，其下木伤。
支克干	截脚	地支克天干。如壬戌（土克水）。地支像砍断天干的脚，天干力量受损严重。如浅水行于燥土，其源易涸。
干支同类	通根	天干与地支为同一五行。如甲寅（木+木）。天干力量得到极大增强，称为“坐实”、“根深蒂固”。

初学者技巧：
•	看到任何一柱，首先判断这五种关系中的哪一种。
•	“通根”和“得生” 是天干有力量的标志；“盖头”和“截脚” 是天干受伤的标志。
### 4.2 异柱干支作用：通根透干
•	核心规则：异柱干支不直接生克，其能量通过“通根”与“透干”关联。
•	通根：天干在地支中有同类藏干支撑。根强则力实，无根（虚浮）则力弱。
•	透干：地支藏干出现在天干，是将潜在能量引发为显性能量的关键。
核心规则：异柱干支不直接生克，但通过 “通根透干” 发生能量关联。

•	通根：天干在地支（尤其是本柱）中有同类的藏干。如甲木天干，见寅木地支，则为甲木通根于寅。
￮	根强：坐长生、禄、旺地（如甲见寅）。
￮	根弱：坐余气、墓库（如甲见未，未为木库）。
￮	无根（虚浮）：地支完全无其五行。无根之干，如水上浮萍，虽有若无。
￮	典籍依据：《滴天髓·通神颂》：“根在苗先，实从花后。” 明确指出了地支（根）是天干（苗）力量的根本来源。

•	透干：地支中的藏干，在天干上出现。如地支寅（藏甲、丙、戊），若天干见甲，则为甲木透干。
￮	意义：透干，是将地支中隐藏的能量“引发”到台面上来，成为显性的、主动参与生克的力量。
￮	典籍依据：《子平真诠·论阴阳生死》：“然透干之力，较藏支为显；藏支之力，待透干而用。” 精辟地阐述了透干的机制。
### 4.3 藏干规则与“寻伏”
藏干是地支能量的“仓库”，需要“钥匙”（透干或岁运引发）才能打开。

1.	藏干的力量等级
•	本气：力量最强，代表该地支的主要属性。通常是该地支的“禄”或“旺”位之干。如寅中甲木。
•	中气：力量次之。通常是该地支所合化的五行或生本气的五行。如寅中丙火（木生火）。
•	余气：力量最弱，是上一季五行能量的残留。如寅中戊土（承接上一季丑土之余气）。
•	典籍依据：《渊海子平·论地支藏遁》完整列出了各地支所藏之干，为后世法则之源头。

2.	透干的规则
•	任何地支中的藏干，都有可能在天干上“透”出来。
•	透出后，该藏干就从一个被动的能量库，变成了一个能主动与其它天干发生生、克、合、冲的“参与者”。
•	案例：八字：甲寅 丙寅 庚午 戊寅。这个八字地支木极旺，年干甲木、月干丙火（寅中藏丙），都是地支寅中藏干透出的结果。这些透干，将地支木火的气势引发到了天干，形成了木生火、火克金的强大攻势。

3.	“寻伏”之法（高级应用）
•	此法用于分析天干在地支中是否有根基，或寻找命局中潜在的能量。
•	寻根：看天干某五行，在地支中是否有它的“根气”。若无，则为虚浮。
•	待用：如果命局急需某种五行，它虽未在天干透出，但安静地藏于地支之中，待大运、流年天干将其“引导”出来（即透干），便能发挥作用。这便是“待时而用”，是判断应期的重要方法。
•	典籍依据：《五言独步》：“有病方为贵，无伤不是奇，格中如去病，财禄两相随。” “去病”的过程，往往就是通过岁运将地支中能“治病”的五行透出或引动。

## 第五章：常见误区与终极辨析

本章旨在根除初学者在应用时，最后可能残留的混淆点。

误区	问题分析	正解与典籍依据
1. 忽略天干五合的优先性	先论天干生克，后论天干五合，导致五行属性判断错误。例如，先论甲木克己土，而忽略了甲己合化土的可能性。	正解：天干作用，合冲优先。 必须首先判断紧贴的天干五合与四冲。合化成功则五行重定；合而不化则相互羁绊，其原有生克能力大减。《渊海子平·论十干合》开篇即论合化，其重要性在生克之先。
2. 异柱干支能直接生克	认为年干庚金能直接克时支寅木。	正解：异柱干支不直接作用。 其能量通过“通根透干”联系。庚金欲克寅木，需寅中甲木透干方可。《三命通会·论人元司事》 专论地支藏干（人元）需透干（出台）方能大用。
3. 将“克”误称为“冲”	认为丁见癸、壬见丙是相冲。	正解：天干相冲仅甲庚、乙辛、丙壬、丁癸四组。 其余皆为“克”。冲是方位对冲、同性相斥，主剧烈变动；克是五行制约，主持续压力。《渊海子平·论十干》 在论及天干关系时，生、克、合、冲是截然不同的概念，不容混淆。
4. 忽略全局合会，强调局部关系	在“寅午戌”三合火局中，仍强调寅午半合、午戌半合，或寅亥六合等。	正解：全局性合会 > 局部作用。 三合局一旦成立，其力量具有压倒性和整体性，内部地支的局部关系被融合、吸收，不再具备独立分析的价值。应视其为一个强大的“火能量团”。《滴天髓·同心》：“一方之气，不可不顾也。”
5. 认为“隔干完全不作用”	完全忽略隔干的存在，认为年干与日干毫无关系。	正解：隔干不直接作用，但中间干至关重要。 它起到“通关”或“阻隔”的决定性作用。分析隔干关系，本质是分析一个连续的作用链条。《子平真诠·论十干合而不合》 中论合化条件时，涉及隔干的影响，实则承认了其间接作用。（前提是这些天干未被紧贴的合冲关系所羁绊）
6. 认为“合”必为吉，“冲”必为凶	见到天干五合或地支六合就断好事，见到相冲就断坏事。	正解：合与冲皆是现象，吉凶取决于喜忌。 用神被合绊（如忌神合克用神）为凶，忌神被合绊为吉。用神被冲伤为凶，忌神被冲去为吉。《滴天髓·恩怨》：“两意情通中有媒，虽然遥立意寻追…有情却为人所忌，无情反是却为奇。”
7. 认为“虚浮”天干完全无用	认为无根的天干在命局中可以忽略。	正解：虚浮的天干代表一种意象和潜在的可能性。 它在岁运遇到根气（通根）时，会突然变得有力，从而发挥作用。它始终是命局的一个组成部分。《五言独步》：“有病方为贵，无伤不是奇。” 虚浮之病，待运而愈，正是应期所在。
8. 分析顺序错误，先论生克后论合冲	先看生克，再看合冲，导致结论错误。	正解：特殊关系优先。 必须先论天干五合四冲、地支三会三合六合刑冲，因为这些关系会改变五行属性（如合化）或完全牵绊住双方（如合绊），使其无法再参与普通的生克。生克是常态，合冲是变态，变态优先于常态。此原则贯穿于所有经典案例的分析中。
9. 地支仅凭五行论生克	认为寅木与午火相邻，寅就能生午。	正解：地支主静，必须凭借刑、冲、合、害等特殊关系，才能产生实质的生克力量。 寅午有半合关系，故能强效相生。若寅与巳，无特殊关系，则仅有微乎其微的气势之生。《渊海子平·论地支》 通篇都在论述地支通过刑冲合害而作用，而非简单五行生克。
10. 合化成功即是好事	见到合化成功便断为吉。	正解：合化的吉凶，完全取决于其对最终喜忌的影响。<br>- 吉：忌神被合化掉，或合化为喜神。<br>- 凶：喜神被合化掉，或合化为忌神。<br>- 核心是看结果净化了命局，还是污染了命局。
11. “墓库”被合化后的特殊性	认为当地支辰（水库）参与合化后，其库中收藏的五行就完全消失了。
	正解：墓库被合化后，其作为“库”收藏杂气的功能会大幅减弱，但并非完全消失。 在特定岁运引动下，仍可能发挥作用，但力量已不如前。例如寅午戌合火，戌土中的辛金几乎被烤化，难有作为。



## 第六章：综合实战演练

我们使用一个典型案例，来完整演示新流程。
八字：壬戌 壬寅 庚午 丙戌

### 第一步：扫描全局特殊关系（直抓核心）
•	扫描天干：发现年干壬水与时干丙火，构成 丙壬冲 (属于天干四冲)。
•	扫描地支：发现月支寅、日支午、年支戌、时支戌，满足 寅午戌三合火局 条件。

### 第二步：格局重构（能量重定）
•	首先处理天干四冲：丙壬冲，水火相战。但二者均非日主，且力量强弱需看地支支持。
•	重点处理地支三合局：
￮	判断合化：化神为火。
▪	是否得令？生于寅月（立春后），木旺火相，火有气。
▪	是否透干？时干丙火透出！
▪	∴ 寅午戌三合火局合化成功！
￮	执行重构：
i.	五行重定：月支寅木、日支午火、年时支戌土，全部重定为 “火” 的组成部分。
ii.	干支重评：
•	月干壬水，坐下的寅木被化为火，壬水被严重截脚，虚浮无力。
•	年干壬水，坐下的戌土被化为火，同样被严重截脚，虚浮无力。年时壬丙冲，因壬水虚浮无根，如杯水车薪，反被旺火蒸干，丙火七杀无损而愈旺。
•	日主庚金，坐下的午火成为合火局中心，被旺火猛烈克制。
•	时干丙火，得整个合局之助，七杀之力达到顶峰。
iii.	力量聚焦：不再单独论寅木生午火，也不再论年时两戌土。整个命盘的核心就是一团熊熊烈火，庚金日主置身于熔炉之中。

•	重构后命局认知：全局一片火海，庚金被旺火围克。天干丙壬冲，以丙火全胜、壬水惨败告终。

### 第三步：定日主旺衰（于重构之基）
•	在火海格局中论庚金：
￮	得令？ 否。月令之气已被火局主导。
￮	得地？ 否。地支无金之实根，反被烈火焚烧。
￮	得势？ 否。天干无助（两个壬水虚浮，无法制火救金）。
•	最终结论：日主极弱，弱至极处，需考虑是否为 “从杀格” （从火）。

第四步：明喜忌用神（战略决策）
•	此局合化彻底，火势炎炎，更符合“从杀格”之象。
•	∴ 喜神为木、火、土（能生火、顺火者）；忌神为金、水（来逆势反抗）。

### 第五步：细析作用关系（微观侦查）
•	天干：
￮	丙壬冲（已分析，火胜水败）。
￮	时干丙火（喜神）克日主庚金：七杀攻身，在此局中为顺势之克。
•	地支：所有力量均已融入火局，内部不再区分任何局部关系。
•	关键点：年、时两戌，并非争合，而是共同为火局提供强大、稳固的根基。
